# am-clients

Client-only Astro site for direct-link interactive presentations and review assets.

## Current Direction

- Keep the site static-first and lightweight.
- Use reusable project entries under `/projects/<slug>`.
- Treat Cloudflare Pages as the future deployment target, but leave deployment wiring and domain choice for a later pass.

## Local Development

- Install dependencies with `npm install`
- Start the site with `npm run dev`
- Open the first presentation at `/projects/groh-fde1`
